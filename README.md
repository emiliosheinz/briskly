## :wrench: Running locally

- Go to the root folder
  ```
  cd briskly
  ```
- Create a `.env` file based on  `.env.example`
- Fill in all environment variables
- Install all dependencies
  ```
  yarn
  ```
- Setup the database with Docker
  ```
  docker-compose up -d
  ```
- Run the dev server
  ```
  yarn dev
  ```
